## Electron Example

### 安装
```
  git clone https://github.com/ounana/electron-example.git
  cd electron-example
  npm install
```
### 启动开发
```
  npm start
  npm run estart
```
### 打包构建
```
  npm run build
  npm run package
```
### help
```
  electron-packager <sourcedir> <appname> --platform=<platform> --arch=<arch> --out <out> 

  <sourcedir> 资源目录
  <appname> 默认pageage.json中 name
  --platform -> darwin | linux | mas | win32
  --arch -> 64
  --overwrite  强制覆盖
  --app-version -> 0.0.1

```
